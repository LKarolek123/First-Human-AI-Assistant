# XO AI Assistant Vision

XO is a Human First desktop AI assistant for Windows.

It is not a chatbot and not another GPT wrapper. The long-term goal is to become the user's central interface to the computer: voice-first, local-first, private, calm, and able to execute useful actions with clear consent.

## Product Priority

1. User wellbeing
2. Privacy
3. Speed
4. Convenience
5. Automation
6. Visual polish

## Principles

- XO supports instead of judging.
- XO helps the user recover calm and make decisions.
- XO explains what it remembers and lets the user edit, export, disable, or delete memory.
- XO asks before risky operations.
- XO works locally by default and uses the internet only when necessary.
- XO is short, warm, natural, and not performatively enthusiastic.

## Core Modules

- Voice First: wake word, STT, TTS, natural conversation.
- Agent: multi-step planning, queues, confirmations, action execution.
- Memory: short-term, long-term, projects, people, preferences, goals, habits, knowledge.
- Plugins: filesystem, browser, calendar, email, Spotify, GitHub, VS Code, Notion, terminal, weather, maps.
- Human Growth: mood, goals, reflection, motivation, wellbeing, healthy habits.
- Productivity: projects, deadlines, habits, free-time suggestions, proactive next steps.
